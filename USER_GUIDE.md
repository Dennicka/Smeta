# USER_GUIDE

## Базовый поток
1. Создайте клиента.
2. Создайте объект.
3. Создайте проект.
4. Добавьте помещения.
5. Выполните расчёт.
6. Сгенерируйте документы и финализируйте их.
7. Зарегистрируйте оплаты.
8. Делайте регулярный backup через Settings.

## Где лежат данные
- База: `~/Library/Application Support/Smeta/smeta.sqlite`
- Backup: путь выбирается пользователем в диалоге сохранения.
