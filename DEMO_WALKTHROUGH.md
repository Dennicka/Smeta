# DEMO_WALKTHROUGH

1. Запустите приложение и дождитесь seed данных.
2. Откройте `Projects` и выберите демонстрационный проект.
3. В `Calculation` выполните расчёт.
4. В `Documents` создайте/финализируйте документы.
5. В `Invoice editor` создайте Faktura.
6. В `Payments` внесите частичную оплату.
7. В `Settings` выполните backup/restore для проверки сохранности данных.
